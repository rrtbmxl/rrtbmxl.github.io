R教程整合

March 16, 2019 in R

还没有想好要写个什么样的开头，所以就先不写了。

想弄这么个东西，目的主要有两个。一方面，把自己会的东西以教程的形式写出来，能让自己把知识掌握得更牢固；另一方面，也能督促自己不断地去学习新知识。虽然心里明白，并没有人会在乎我作了什么承诺，但自己立下的承诺，总会有一些约束力的。当然，如果能对别人也有一些帮助，那自然是更好了。内容方面，包括我目前会的，和我将来想学的，以及我自己的一点点心得体会，所以这个教程会有很强的个人色彩，具体内容可以看后面暂定的大纲。另外，有几点需要提一下：

  • 术语的翻译。对于已经有约定俗成的译法的术语，当其第一次出现时，会附有对应的英文，如果这一术语我没有能力解释，就会附上相应的参考资料（以维基百科为主）。如果某一术语我翻译不出来，或者翻译成中文会在很大程度上失去原来的味道（如tidy data），我就不进行翻译了，直接使用英文。另外，对于一些符号，当其第一次出现时，我也会附上相应的英文，如~（tilde），`（backtick）。我在读英文教程时，总被不知道某一符号怎么念而困扰，所以我猜别人也会有这种苦恼吧。

  • 更新的应对。无论是R本体，还是那些包，都是会不断地更新的。有些代码，可能只在某些版本下才能运行，更新之后就不行了。比如，David Robinson在16年情人节的时候写了一个画动态心形图的教程，但现在就运行不了了（从这一点来看，也不建议去买翻译过来的书去看，很有可能，一本书翻译过来之后，里面的一些代码就无法运行了）。如果时间允许的话，我尽量保证我的教程里的代码都是在最新版本下可运行的。

  • 教程的引用。参考的教程统一附到最后，对于暂时没找到印证资料的，会使用“似乎”、“大概”、“可能”等词汇进行描述。
